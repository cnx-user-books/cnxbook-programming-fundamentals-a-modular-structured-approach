This is a flash card activity. Here is a list of the terms and definitions on each card.

Card 1:
Term: flowcharting
Definition: A programming design tool that uses graphical elements to visually depict the flow of logic within a function.
Card 2:
Term: process symbol
Definition: A rectangle used in flowcharting for normal processes such as assignment.
Card 3:
Term: input/output symbol
Definition: A parallelogram used in flowcharting for input/output interactions.
Card 4:
Term: decision symbol
Definition: A diamond used in flowcharting for asking a question and making a decision.
Card 5:
Term: flow lines
Definition: Lines (sometimes with arrows) that connect the various flowcharting symbols.
Card 6:
Term: control structures
Definition: Mechanisms that allow us to control the flow of execution within a program.
Card 7:
Term: sequence
Definition: A control structure where you do the items in the sequence listed.
Card 8:
Term: selection
Definition: A control structure where you select between two or more choices.
Card 9:
Term: iteration
Definition: A control structure that allows some lines of code to be executed many times.
Card 10:
Term: branching
Definition: A control structure that allows the flow of execution to jump to a different part of the program.
Card 11:
Term: structured programming
Definition: A method of planning programs that avoids the branching category of control structures.
